NKCamera
=============
http://nanoka.wpcloud.net  

機能
-----
Objective-cでのカメラ機能でよく使うものをサンプルとしてまとめました  
Swiftでの実装はこちら:https://github.com/nanokanato/NKCamera_Swift  
  
・露出を合わせる  
・フォーカスを合わせる  
・撮影画像の保存  
・カメラの切り替え  
・フラッシュ切り替え
